About

In the lack of an "Accept all" button for gift requests in FarmVille I decided to make this extension for the Chrome browser and named it FV Extender.

The ideas for improvements and features are however many so this will not be the only feature. That is why it got the name FV Extender and not just "Accept all extension for FV".

I sincerely hope that Zynga and Facebook in time will adopt the features instead. Because FV Extender cannot always give you the same smooth user experience as a direct implementation.

It is not intended that any of the features will in any way ruin the experience for your neighbors in FarmVille. Instead the features will focus on doing the opposite for both you and your neighbors.

Is it safe to use?

When you install the extension in Chrome you will first get a pop-up which states what the extension wants access to. All it will need to access is your view of Facebook/FarmVille so it can enhance this. So do not fear it will delete files or plant a virus on your hard drive. It will not. This is how the extensions in Chrome are designed. Chrome extensions cannot ask for this kind of access at all.

How do I get started

If you already have the Google Chrome browser installed and know how to install an extension you can just head over to the Download page and be on your way in a few minutes.

Otherwise you can start by reading the User guide where the complete installation procedure is described.
