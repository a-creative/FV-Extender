About

In the lack of an "Accept all" button for gift requests in Farmville I decided to make this extension for the Chrome browser and named it FV-Extender.

 

The ideas for improvement are however many so this will not be the only feature. More will come. That's why it got the name FV-Extender and not just "Accept all extension for FV". That also made the name simpler.

 

I sincerely hope that Zynga and Facebook in time will adopt the features instead. Because FV-Extender tries to hack the system and is as such not always able to give the same smooth user experience as a direct implementation.

 

It is not intended that any of the features will in any way ruin the experince for your neigbors in Farmville. Instead the features will focus on doing the opposite for both you and your neigbors.

 

Is it safe to use?

When you install the extension in Chrome you will first get a popup which states what the extension wants access to. All it will need to access is your view of facebook/farmville so that it can enhance this view. So don't be affraid it will delete files or plant a virus on your hard drive. It will not. That's how the extensions in Chrome are designed. Chrome extension can't ask for that kind of access at all.

 

How do I get started

If you already have the Google Chrome browser installed and know how to install an extension you can just head over to the Download page and be on your way in a few minutes.

 

Otherwise you can start by reading the User guide where the complete installation procedure is described.
