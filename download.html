System requirements

  • Google Chrome web browser. Get it here
  • Tested on Google Chrome version 6.0.472.63 but should work on some earlier versions too.

Latest release

Left click the image below to download. The downloaded .crx file will start the installation in Google Chrome when run/activated. Read more about the installation in the User guide.

When initially installed FV-Extender will automatically update it self.

Below you will always be able to see information concerning the latest release.

New features

  • Adds and "Accept all" button above the list with FarmVille gift requests in Facebook.

Bugs fixed

  • Initial relase. No bugs reported yet.

Known issues

  • The "Accept all" button does sometimes not show up. Please reload the page if this happens.
  • Ignoring a gift just before clicking "Accept all" does not work. Please reload the page if this happens, and click the "Accept all" button again.
  • The "Accept all" button will only appear at the general list of gift requests in Facebook and not at the games overview which was introduced by facebook in September 2010.
