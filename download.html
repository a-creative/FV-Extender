System requirements

  • Google Chrome web browser. Get it here
  • Tested on Google Chrome version 6.0.472.63 but should work on some earlier versions too.

Latest release

Left click the Download image below to go straight to the extension's page on googles app store. Then click the install button and you are done.

When initially installed FV Extender will automatically update itself.

Below you will always be able to see information concerning the current release and previous releases.

Follow the project on Twitter to stay updated all the time.

Version 3.0(current release)

New features

  • FV Extender can now automatically return a gift when accepting one.
  • Cow sound when the proces is done.
  • Sheep sound when the proces is aborted by FV Extender.
  • "Granted wish", "Thank you" and "Don't resend" requests are now accepted without returning gifts.
  • "New neighbor" requests are skipped for manual processing.
  • Added in 3.0.2: The toolbar promotion screen is now handled as an error screen and FV Extender redirects you to the list of game requests so the processing of requests can continue.

Known issues

  • The auto gift back feature is still experimental but I decided to give it to you anyway. I'm constantly working on making it completely stable. If you experience that the proces stops in the middle of it all please reload the page and click the "Accept all" button again. If it is a constant problem please report it back to me.
  • If you wonder why some of you game requests just don't get accepted it might be because FV Extender skips some types of game requests automatically. Read more about this in the user guide.
  • FV Extender does currently not run in an incognito window. So for now you will have to play with open cards to use it :-)
    Thank you Roy for reporting this bug in the support forum.
  • FIXED in v. 3.0.1. When you reach your gift box limit it will currently abort say something with [BUSHEL LIMIT]. It correctly abort so you can make room in your gift box, but the message could be better. Will fix this ASAP.
  • FIXED in 3.0.2: The "Oh no It looks like all the bits got lost" error screen has so far not been handled correctly. It should be handled correctly now and will return you to the list of game requests so the processing of game requests can continue.
  • FIXED in 3.0.2: Some of you couldnt see the popup when you clicked the "Accept all..." button. The position of the window was simple wrong. Now it's been changed to be in the middle of the screen. That should work for everybody.

Version 2.0

New features

  • Everything has changed except the position of the "Accept all" button.
  • The gift requests are now accepted much quicker...
  • and much nicer :)

Bugs fixed

  • FIXED: The cycle of accepting of gifts sometimes stop unexpectedly. Please reload the page if this happens. In this case the "Accept all" button will also be invisible.
  • FIXED: Ignoring a gift just before clicking "Accept all" does not work. Please reload the page if this happens, and click the "Accept all" button again.

Known issues

  • Sadly the "Accept all" button still sometimes not show up. I'm not sure this problem can be fixed without doing some serious changes. For now you will have to reload the page if this happens. One reload usually does it.
  • The "Accept all" button will only appear above the general list of gift requests on Facebook and not at the games overview which was introduced by Facebook in September 2010. This issue has a very low priority on my list. So don't expect it to change in the near future.
