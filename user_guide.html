System requirements

  • Google Chrome web browser.
  • FV Extender is tested on Google Chrome version 8.0.552.215 but should work on some earlier versions too.

Installing the Google Chrome web browser

Google Chrome is one of the browsers that runs Farmville(and other games made by Zynga) most smoothly. That is why this extension/plug-in is made for Google Chrome and not eg. Firefox.

Google Chrome is by far also the fastest and easiest browser to install. Follow Google's instructions to install it.

You can find more information about what Google Chrome is by reading Google's own presentation of Chrome.

Downloading, installing and testing FV Extender

Installing an extension in Google Chrome is also very easy and can be done in a few minutes.

The easiest way to start the installation of FV Extender is to click the download link using Google Chrome as a web browser. There is a link to the download page at the bottom of this paragraph.

Next you will be redirected to the extension's install page at the google extension store.

Next a 'Confirm installation' pop-up will show.

This pop-up window is important to notice when you install extensions within Chrome. Because it's here you get informed about what parts of your system the browser needs access to. So if the text show eg. "Access to delete files on your hard drive" then you better not install the extension. FV Extender does however not need that kind of access. So don't worry about getting your computer infested with virus because of FV Extender.

When you have read and understood the 'Confirm installation'-pop-up you can click "Install" and another pop-up shows in the top right corner when the installation is done. This usually happens within seconds.

Finally to test if FV Extender works you can log into Facebook and have a look at your list of game requests. You should hopefully see a new button with the text 'Accept all' above the list of game requests from FarmVille(if you currently have any).

Try clicking it and prepare yourself for the magic of FV Extender.

Disabling FV Extender

You can disable FV Extender if you for some reason find it annoying.

To learn how to disable an extension in Google Chrome follow Google's own instructions.

Stay up-to-date

Follow us on Twitter to get all updates on fixed bugs and new features.

The extension automatically updates itself at every new version. This does not happen as instantly as tweets arrive. So to force an update:

  • Go to this URL in your browser: "chrome://extensions"
  • Enable "developer mode".
  • Finally click the "Update extensions now" button.

Using FV Extender / features(3.0)

If you just hate reading here is a shortcut to a feature demonstration on Youtube: http://www.youtube.com/watch?v=CqXibcYGnfs

You will now get a walk through all the features of FV Extender and how they are used.

  • The "Accept all" button
    This button only shows when there are visible game requests in Facebook related to Farmville.

    If you cannot find the button this is the exact link to the page you will find it on: http://www.facebook.com/reqs.php#confirm_102452128776_0. Try reloading the page if you still don't see the button.

    Clicking the button show then options window. From here you basically decide what you want to do with the pending game requests. In the future this window will give you the options to only accept some requests. In version 3.x you however "only" have three main options:

    Accept: Click the "Accept" button to quickly accept all your game requests without returning a gift. This is great if you want to surprise your neighbors with gifts instead of just returning what they got for you. It's also much faster than when you want to return gifts.

    Accept & return gift: Click the "Accept and return" button to accept the pending game requests and at the same time send a gift back. Many of you send out gifts to get the same back. FV Extender now honours this by not only making it possible but also gives you the posibility to decide what text message to send with the return gift. I advice you to only enter a message if you really got something on your mind, because all other users of FV Extender will be forced to read this message. This does however not count for a message including the default text "This gift was returned using FV Extender".

    Just help: Click the "Just help" button to only proces requests related to helping. This way you can quickly and maybe more often get your neigbors the building parts they need and can save the actual gifts for later.

    The magic starts when you've clicked one of the three buttons just mentioned. All your game requests are then accepted one by one and you will see a status window so you know when it's done. Meanwhile you can relax and have a cup of coffee.

    A muuh(cow) sound will play from your speakers when all game requests are accepted. So if you turn up the volume you can actually relax in a completely different room while FV Extender is at work for you.

    Gift box protection. FV Extender will abort the proces if you reach your gift box limit during the accepting of game requests. You can then click "Accept all" again when you have made some room in your gift box.

    Bushel protection. FV Extender handles bushel gifts in the same way. Wouldn't it be foolish to loose something you have asked for? FV Extender aborts the process when you have reached the bushel limit.

    A meeeh(sheep) sound will play when the proces is aborted by the system. So when you hear strange animal sounds coming from your computer it's just FV Extender that needs your attention.

    Disabling sound: In the initially shown option window It is possible to disable the cow and sheep sounds if your boss does not like them :-)

    "Shovel" requests are at the moment not automatically accepted: For now you will have to accept these manually and pick the gift you want for sending a shovel to your neighbor.

    Requests with a personal message are also skipped.     This way you don't miss any of these messages. Some people write some really nice messages and you don't want to miss these. Right? As mentioned earlier on game requests with the message: "This gift was returned using FV Extender" are NOT skipped but instead automatically accepted.

    Returning "Thank you gifts" for "Thank you gifts". In the initially shown option window it is possible to enable/disable this feature.

    "Don't resend" and a like messages usually means that a neigbor want to give you something but don't want it back. FV Extender accepts these game requests without sending a gift back. Don't you think your neigbors(and you) would really like that.

    Granting a neigbors wish usually does not mean you just want the same back. Therefore these game requests are accepted without returning gifts.

    New neighbor requests sometimes come from old Facebook friends. Therefore these requests are skipped so you always knows when this happens.



    More features
    There are no more features for now, but the "Accept all" button is a great starting feature. Stay tuned for more.
